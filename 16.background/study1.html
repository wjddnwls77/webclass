background

태그의 공간에 배경색 또는 배경이미지를 나타낼 때 사용하는 css 속성

  • background-color(transparent) ->배경색상을 입력할 때 사용하는 속성
  • background-image(none) ->배경이미지를 입력할 때 사용하는 속성
  • background-repeat(repeat) ->
    • 입력된 배경이미지의 가로세로반복속성을 제어하는 속성
  • background-posistion(0px 0px) ->입력된 배경이미지의 위치를 x/y축값으로 제어하는 속성
  • background-size(auto auto) -> 배경이미지의 가로세로사이즈를 지정하는 속성
  • background-blend-mode -> 배경색 또는 배경이미지들의 혼합옵션을 지정하는 속성
  • background-attachment(scroll) -> 배경이미지의 붙임속성을 지정하는 속성

background-position

background-position 속성은 기존 좌상단을 (0,0)좌표로 생각 했을 때 좌표체계에 따라서 픽셀단위로 수치를 입력하여 배경이미지를 변경시키는 방법이 있다. 하지만 그런 방법만 통해서 배경이미지를 선택영역의 가로기준 중앙 세로기준 중앙에 배치하려고 하면 계산하는 과정이 복잡하기 때문에 키워드 방식으로도 배경이미지를 배치 할 수 있다,(left/right/top/bottom/center)

키워드 입력 방식으로만 배경이미지의 위치를 변경하기에는 표현에 제약이 많이 따르기 때문에 %단위를 사용해서도 배경이미지를 배치할 수 있다. 이 때, 좀 더 세부적인 위치변경을 하기위해서 calc()라는 css상의 계산기를 활용할 수 있다.

calc() 괄호 안에는 +(더하기).-(빼기).*(곱하기)./(나누기)기호를 통해 계산을 할 수 있다. 이 때 더하기와 빼기는 부호 앞뒤로 반드시 띄어쓰기를 해서 표현해야 하며, 곱셈 나눗셈은 띄어쓰기를 하지 않아도 된다.

background-size

background-size:(가로사이즈) (세로사이즈) 속성을 통해서 배경이미지의 가로와 세로사이즈를 지정할 수 있다.

이 때, 가로사이즈를 픽셀이 아닌 %를 사용할 수 있다.

그리고, auto라는 키워드를 통해서 원본이지미가 갖고 있는 비율계산은 자동으로 적용해 줄 수 있다.

  • cover -> 원본 이미지의 가로세로 비율을 유지하면서 선택한 태그의 공간을 빈틈없이 가득 매꾸는 방식 (이 때, 배경이미지가 잘려나가도 신경쓰지 않는 방식)
  • contain -> 원본 이미지의 가로세로 비율을 유지하면서 선택한 태그의 공간을 최대한 가득 메꾸는 방식(이 때, 배경이미지가 절대 잘려나가지 않는 선에서 사이즈를 조절하는 방식)
