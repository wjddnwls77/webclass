transform

transform은 변형이라는 뜻으로 주로 css에서는 animation을 만들 때 사용한다.

부분적으로 레이아웃을 위해 사용하기도 한다.

transform - translateX(픽셀 값, % 값)

transform에서 translate는 픽셀 값과 % 값을 둘다 입력 가능하다. 이 때 % 값은 선택된 태그의 너비나 높이를 기준했을 때의 비율이다.

기본적으로 transform속성은 박스형태(block,inline-block,flex)한테만 적용가능하다.
