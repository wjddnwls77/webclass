position

일반적인 레이아웃이 아닌 특수한 경우에 자주사용되는 레이아웃 방식

  • relative
  • absolute
  • fixed

position-relative

  • 현재 태그가 배치된 공간을 기준으로 이동하는 방법 (left/right/top/bottom)사용
  • (left:100px) -> 태그의 왼쪽 테두리 면에서 안쪽으로 100픽셀 떨어진 곳으로 태그가 이동
  • position:relative 속성을 사용해서 태그가 이동하는 경우 원래 배치되어 있었던 공간을 그대로 차지하며, 다른 곳으로 이동한다고 해도 다른 태그의 배치에 영향을 미치지 않는다.

position:absolute

position:absolute가 적용된 태그는 자신을 포함하는 상위태그 중에서 가장~~~~가까운 position이 적용된 태그(->css 상에서 position 속성을 사용한 태그)의 영역을 기준으로 배치된다. (이때 body 태그는 기본적으로 position 상태가 항시 적용이 되어 있다.)

position:absolute가 적용된 태그는 공중에 띄어져서 배치되기 때문에, 다른 태그의 레이아웃에 영향을 미치지 않는다.
