css

css는 html태그를 스타일링하는 styel 언어

방식

  • inline-style(element-stlye)-직접선택방식

    태그를 직접 선택하여 시작태그에 style이라는 속성을 해서 스타일을 입력하는 방법

  • 내부스타일시트

    heda태그안에 style태그를 작성하여 스타일을 입력하는 방법

  • 외부스타일시트

    html파일 바깥에 별도의 css파일을 생성해서 스타일을 입력하는 방법

css기초문법

css는 태그를 선택해서 스타일을 입력하는 언어이다. 그렇기 때문에 선택하는 부분과 스타일이 입력되는 부분으로 나뉜다.

선택자{스타일속성:스타일속성값;스타일속성:스타일속성값}

선택자는(태그명,클래스명,아이디명)으로 선택할 수 있다.

;(세미콜론)은 구분자로 사용된다.
