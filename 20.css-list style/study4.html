transistion(전환)

전환 속성을 사용해서 간단한 애니메이션을 구성할 수 있다.

전환속성은 변하기 전스타일에 미리 설정 해야한다.

전환속성은 중간스타일을 원하는 시간동안 원하는 시간 뒤에 원하는 속도로 표현하는 css 속성이다.

전환속성은 중간수치를 표현할 수 있는 css속성에 대해서만 적용 가능하다.
(예:width:300px ->width:500px (전환속성 설정가능)/ )
(예:diplay:block -> display:flex (전환속성 설정 불가능)/) = 형태가 바뀌는 것은 명확한 계산가능한 중간수치를 표현하는 개념이 아니기 때문에 전환속성을 사용해도 적용되지 않는다.
