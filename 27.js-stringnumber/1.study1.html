자바스크립트 - javascript

자바스크립트언어를 통해서 홈페이지의 기능을 추가해줄 수 있다.

html은 정적인 언어이다. 그렇기 때문에 유동적인 컨텐츠를 표시해 줄 수가 없다. 이를 위해 자바스크립트 언어가 동적으로 작동하여 유동적인 컨텐츠를 보여준다.

변하는 컨텐츠를 보여주기 위해서 자바스크립트가 표현하는 6가지 데이터 형태를 공부해보자.

자바스크립트도 기본적으로는 위에서 아래 왼쪽에서 오른쪽으로 해석한다.

자바스크립트에서 명령문뒤에는 명시적으로 ;(세미콜론)을 써주어야 한다. 하지만 줄바꿈을 하는 경우가 대부분이고 줄바꿈을 하면 자동으로 ;(세미콜론)이 붙은 것처럼 해석되기때문에 생략가능하다.

문자데이터 / string

문자데이터를 표현하기 위해서는 앞뒤로 ""쌍따옴표 혹은 ''홑따옴표를 사용할 수 있다.그리고 ``백틱으로도 표현할 수 있다. 백틱은 - 문자데이터가 보장이되지만 쌍따옴표와 홑따옴표는 문자데이터가 보장이 안됨.

문자데이터 안에 들어가는 컨텐츠 내용으로 쌍따옴표 혹은 홀따옴표가 사용될 수 있기 때문에 문자데이터의 표식은 3가지 형태가 존재한다.

*****문자데이터는 +(덧셈연산자)를 통해서 서로 이어줄 수 있다.*****

숫자데이터 / number

사칙연산이 가능하다

수학연산은 덧셈 뺄셈보다 곱셈 나눗셈이 우선계산된다. 덧셈 뺄셈으로 우선계산하고 싶다면 명시적으로 ()괄호를 사용하여 묶어준다.

문자데이터와 숫자데이터를 더하면 서로 이어지면서 결과적으로는 문자데이터로 된다.

  • +(덧셈)
  • -(뺄셈)
  • *(곱셈)
  • /(나눗셈)
  • %(나머지연산)
